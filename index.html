Projects

Major Projects

Skyline Query in DBMS

I worked on this project during my graduate studies in the CSE 510: Database Management System Implementation course, in the Spring 2021 semester. This was a group project; each member had to implement one algorithm for evaluating the Skyline of a given table in the DB, and one other feature. I designed the Nested Loop algorithm for Skyline evaluation and the Hash Join algorithm for improving the performance of the existing Join query. These features were developed on Minibase, a skeletal RDBMS. We used SQL and Java to perform these tasks.

Chargify

I worked on this project with a team of 3 people participating in the Devpost Square Hackathon. We developed a responsive web application to enable users to book reservations for charging their EVs at a nearby charging station. The users could look for a charging station on a map, decide on the time of booking, and see all the reservations they have previously booked. The payment was handled using Square Payments API. I developed the client-side components of the application in a mobile-first manner, using React and Redux.

Face Recognition in Videos

This was the capstone project for my undergraduate degree. The goal was to create an application which can detect and identify a person's face in a video stream. This was a group project where I worked on the module to detect faces in the frames of the video input. For this, I trained a Convolutional Neural Network model created using Keras and Tensorflow in Python. This module had an accuracy of 99% for a 16 fps video stream.
